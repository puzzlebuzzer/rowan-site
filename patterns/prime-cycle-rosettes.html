Prime Cycle Rosettes

What happens when a repeating color cycle is interrupted at prime lengths?

Plain language

Each circular rosette in this piece follows the same rainbow color sequence. But instead of repeating after the same number of colors each time, the sequence stops at different prime numbers. The rosettes use these cycle lengths: 2, 3, 5, 7, 11, 13, 17, 19, 23. Because prime numbers do not share divisors, each rosette breaks the cycle in a slightly different way.

Structural description

The underlying structure is a radial rosette with evenly spaced petals. Each petal advances one step along a cyclic color sequence. When the cycle length is prime, the sequence distributes differently around the circle compared to composite lengths. The visual result is an interaction between radial symmetry and modular interruption.

What I was experimenting with

I wanted to see how prime cycle interruptions manifest visually when mapped onto symmetric geometry. The color generator was red1 → orange1 → yellow1 → green1 → blue1 → violet1 → red2 → … and each rosette used a longer prefix before repeating.

Observations

Different primes create noticeably different global rhythms even though the local rule is identical in every rosette. The larger petals act like a sampling frame for how each prime-length cycle projects onto the six-hue rainbow structure.
