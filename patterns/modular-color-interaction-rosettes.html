Plain language

Each rosette follows a repeating color sequence. But the number of colors used before repeating changes from one rosette to another, so the same generator produces different visual rhythms.

Structural description

The geometry is a radial rosette divided into evenly spaced petals. Changing the cycle length changes the modular relationship between the color sequence and the rosette’s symmetry.

What I was experimenting with

I was exploring how modular arithmetic manifests visually when mapped onto symmetric geometry.

Observations

Some cycle lengths emphasize larger petals while others break the symmetry into smaller fragments. The visual differences become surprisingly pronounced.
